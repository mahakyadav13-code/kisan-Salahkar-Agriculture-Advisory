Hardware Setup Guide

1
Flash ESP32 Firmware

Open hardware/kisan_esp32_sensors.ino in Arduino IDE, set your WiFi SSID & password, and upload to ESP32.

2
Wire Sensors

DHT11 → GPIO 4  |  Soil Moisture → GPIO 34  |  pH Sensor → GPIO 35  |  Power via 18650 batteries.

3
Set Server IP

In the firmware, change SERVER_URL to your computer's IP address (e.g., http://192.168.1.100:5000).

4
Power On & Watch

Power on the ESP32. Data will appear here automatically every 30 seconds!

Wiring Diagram

┌──────────────┐
│    ESP32     │
│              │
│  GPIO 4  ────── DHT11 (DATA pin)
│  GPIO 34 ────── Soil Moisture (AO pin)
│  GPIO 35 ────── pH Sensor (AO pin)
│  3.3V    ────── Sensor VCC (all)
│  GND     ────── Sensor GND (all)
│  VIN     ────── 18650 Battery + (via holder)
│  GND     ────── 18650 Battery − (via holder)
└──────────────┘

DHT11: VCC → 3.3V | DATA → GPIO4 | GND → GND
Moisture: VCC → 3.3V | AO → GPIO34 | GND → GND
pH: VCC → 3.3V | AO → GPIO35 | GND → GND
